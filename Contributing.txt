If you want to contribute follow our Style-Guide.txt

Once you create your theme put in /apps and make the APP NAME A DIRECTORY.The submit a pull request with link to the apps website. And create an INSTALL.txt file in the directory

The tree should look like

|/apps
 |/app name
   |config theme file for app
   |Install.txt


And under "SUPPORTED APPS" in README.txt add the following. Default to the projects website if it exists.
<app name> | <app website /github repo>
Easy

@OwenGrumbles will review the PR.
